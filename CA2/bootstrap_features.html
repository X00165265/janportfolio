Bootstrap Features Used

  • Navbar: Used for navigation at the top of the page.
  • Container and Fluid Container: Used for creating layout containers to hold content.
  • Image: Used for displaying images.
  • Card: Used for creating card components to display information about projects, education, etc.
  • Accordion: Used for displaying module grades in an accordion style.
  • Carousel: Used for displaying career interests in a carousel format.
  • Alert: Used for displaying a success message after submitting the contact form.
  • Navbar Toggler: Used to create a collapsible menu on smaller screens.
  • Button: Used for form submission in the contact section.
  • Form: Used to create a contact form for users to submit messages.
  • Modal: Used for displaying additional content or messages, can be triggered on demand.
  • Carousel Indicators and Controls: Used for navigation within the carousel component.
  • Badge: Used for displaying a badge-like label on the LinkedIn, Email, and Location icons.
  • Tab: Used for creating tabs in the Experience section.
  • Accordion: Used for organizing content in the Education section.
